# Win10TouchTakbar
模拟触摸屏使优车右侧图标间隙
# 关于
此软件由Gitcode中精品资源铺的MultiTouchVista修改版内的驱动安装实现，适用UU远程用户、单纯想给电阻屏使用触摸屏后续间隙更容易按到网络按钮、音量键

原作者仓库在https://gitcode.com/Premium-Resources/f4d04?utm_source=article_gitcode_universal&index=top&type=card&&uuid_tt_dd=10_9921234680-1760767492584-510696&isLogin=9&from_id=148187064&from_link=ca2e1ca0dab7ae7ad7a71061329c094d&tab=md

如果需要使用对于MultiTouchVista软件请回到原仓库

# 用法
下载，解压，打开TouchTaskbar，选择架构（X64为64位，X32是32位）双击打开Install driver.bat，如果显示要求请允许安装

安装完成后在设备管理器中找到“Univer Software HID Device”，禁用再启用即可
